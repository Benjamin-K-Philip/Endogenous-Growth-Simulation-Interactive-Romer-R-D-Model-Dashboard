# Endogenous Growth Simulation Interactive Romer R&D Model Dashboard

 
The simulation represents the Romer endogenous growth model, whereby an economy with given total labour \(L\) generates workers to the production \(L<sub>Y</sub>\) and research and development \(R<sub>&amp;D</sub>, L<sub>R</sub>\) sectors. Capital and labour are deployed by firms in a production to generate an output whereas it is R&amp;D labour that generates new ideas that promote a sustained growth as opposed to Solow model where technology is exogenous. The users manipulate the major parameters such as R&amp;D productivity \(\theta\), markup \(\mu\), interest rate \(\r\), capital intensity \(\alpha\) and labour in real time to observe the equilibrium labour.
